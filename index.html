MEEM Workflow Automation 

What is Workflow?
Workflow is a set of tasks  arranged in a predefined pattern  to carry out a specific procedure. Workflow are present in nearly all organizations. 
What is the Problem?
Usually the workflow process is carried out  using paper memos  with signatures or emails. This is time consuming, inefficient in handling, incapable of producing high level reports etc  
Solution
Our product, MEEM workflow automation, automates such process  in minimal time to make a paper-less environment  using web-based electronic resources. 

Role-based secure access to all particapnts of a workflow is provided. Issues related with Accuracy, Data Management, Security, Tracking are efficiently handled. The System automatically sends email reminders to delayed task holders to expedite process. .
 
Following are some examples of workflows which has been successfully automated using MEEM Workflow automation. 

Promotion System

A customizable web-based automation for faculty or staff Promotion process

It is a workflow system for promoting faculty candidates to a higher rank in a university. The system allows uploading of documents, maintaining research points sheets, selection of external reviewers, to and fro movement of promotion application among different departments, tracking of application with automatic email reminders etc. The system is easy to integrate and customizable online

Events Management System

A customizable web-based automation for Managing Events in any organization 

It is a portal to Create, Modify, Search and Delete Events. Registration of Speakers,Room and Attendants  is carried out through user-friendly online forms. The concerned users are consequently reminded with Email reminders.

Inventory System 

A customizable  web-based automation of Store items for managing items in any store

Process of Issuing, returning, discarding, and adding new items  in a store is automated.High-level summarized reports are made for the Higher authorities to help make decisions 

Budget Administration System 

A customizable  web-based automation for administering Budgets of Projects Grants.  

The projects grants from an external sources in an organization needs to be organized and managed. This web-based automation allow Prinicipal Investigators of mutlitple projects secure logins to manage budget heads and allocate funds for running the project. The system autmatically  provides monthly, yearly and final financial reports to be submitted to the external source. 

Projects Management System 

A customizable  web-based automation for managing project quality

The projects quality need to managed while the project is running. This automation provides timely reports to the center director, reporting of any problem like equipment missing, closing of project etc. .
