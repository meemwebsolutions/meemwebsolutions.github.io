Our Ready made Workflow Automations

Below is a list of out of box customizeable automation of different workflows present in MEEM Workflow Automation Framework

Project Management Workflow

The projects quality need to managed while the project is running. This automation provides timely reports to the center director, reporting of any problem like equipment missing, closing of project etc. .



Clearance System

In this solution, request is raised by employees leaving the organization and applying for clearance of assets under his/her account. The request is forwarded to different departments that can be tracked by the applicant. Once completed successfully, the employee can see the archived application with all needed attributes. If the application need to be revised then it is returned back to applicant with message to perform certain action like returning keys or returning equipments etc.


Travel Request System

In this automation, employees in an organization can create travel requests by filling up forms mentioning employees dependents information, start date, end date, destination places, purpose etc. This travel request if further processed by higher authroities while the applicant can track his application. Once completed the application reaches archived state and is returned back to the applicant. If any other document or information is required from the applicant then it is returned back to him to return again.

School Admission Management System

The system automates the workflow of admission process in school. Online Approvals and rejection, Tracking of application, Reminders notification for expediting process are salient features of this system,


Evaluation System

Employees make an online request by filling out a form that is submitted to immediate boss then to other higher authorities of the organization. 

Purchase Request Automation

Employee can select items, mention quantity and proper justification to place a request of purchase. The request will go through different departments till final approval
